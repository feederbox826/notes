# dedupe tools and similar programs

An opinionated list of dedupe tools and similar utilities, in no particular order

# multi-file/ agnostic
## [scrubbbbs/cbird](https://github.com/scrubbbbs/cbird)
cli image and video dedupe
- similarity & hash dedupe
- cache
- GUI
- zip traversal

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/scrubbbbs/cbird)

## [0x90d/videoduplicatefinder](https://github.com/0x90d/videoduplicatefinder)
video and image similarity dedupe
- cache
- hwaccel
- metadata diff comparison
- multi-file grouping and comparison

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/0x90d/videoduplicatefinder)

## [qarmin/czkawka](https://github.com/qarmin/czkawka) (aka Krokiet)
multi-functional dedupe file/ cleanup
- empty directories
- similar images
- duplicate files
- cache
- GUI
- set reference folders
- no live image preview
- good all-rounder, loses to more specific tools in speed or functionality

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/qarmin/czkawka)

# videos
## [kristiankoskimaki/vidupe](https://github.com/kristiankoskimaki/vidupe)
- SSIM / phash filtering for duplicate images
- adjustable thumbnail modes
- cache
- move/ swap filenames/ delete

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/kristiankoskimaki/vidupe)

# images
## [ermig1979/AntiDupl](https://github.com/ermig1979/AntiDupl)
GUI image comparison with powerful compare
- cache
- dimension/ EXIF/ blockiness comparison
- extremely powerful GUI and quick keyboard shortcuts
- set reference and preferences for folders to delete

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/ermig1979/AntiDupl)

## [Awesome Duplicate Photo Finder](https://www.duplicate-finder.com)
- Powerful search differencing & simple GUI
- no support for modern files (webp/ avif)
- propreitary

[archive](https://archive.org/details/awesome-duplicate-photo-finder)

# tools
## [0x09/resdet](https://github.com/0x09/resdet)
- detect source resolution of upscaled images and videos

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/0x09/resdet)

## [vzhd1701/gridplayer](https://github.com/vzhd1701/gridplayer)
simultaneous multi-video player

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/vzhd1701/gridplayer)

## [google/spatial-media](https://github.com/google/spatial-media)
inject spatial media metadata into VR videos (interactive VR projection in video players)

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/google/spatial-media)

## [hxseven/Remove-Empty-Directories/](https://github.com/hxseven/Remove-Empty-Directories/)
Remove empty directories

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/hxseven/Remove-Empty-Directories)

## [fifonik/FFMetrics](https://github.com/fifonik/FFMetrics)
[VMAF](https://en.wikipedia.org/wiki/Video_Multimethod_Assessment_Fusion) / [SSIM](https://en.wikipedia.org/wiki/Structural_similarity_index_measure) / [PSNR](https://en.wikipedia.org/wiki/Peak_signal-to-noise_ratio) assessment for videos
- how much a [reencode/ transcode/ resize] affected the ***objective*** quality of the video

[archive](https://archive.softwareheritage.org/browse/origin/directory/?origin_url=https://github.com/fifonik/FFMetrics)

## [NVIDIA ICAT](https://www.nvidia.com/en-us/geforce/technologies/icat/)
- image/ video comparison
- overlay/ slider comparison
- persistent zooming
- propreitary

## [Hive Moderation AI detection](https://hivemoderation.com/ai-generated-content-detection)
- pretty decent AI generated image detection
