# what to look out for
- Browser extensions can be a plus if you're isolating stash with a browser profile
- Wireguard usually outperforms OpenVPN and I prefer it as a protocol
- Providers that need additional config per-country can create issues when moving between regions to test geoblocks.

A good starter list of providers is at [gluetun wiki](https://github.com/qdm12/gluetun-wiki/tree/main/setup/providers). [torrentfreak](https://torrentfreak.com/tag/vpn/) is a good resource for which providers have been tested against warrants in court.
